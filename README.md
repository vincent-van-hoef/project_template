# project_template

A skeleton template for analysis projects.

Use this repo as a template. 

To activate the GitHub pages Sphinx static site, go to the Settings of the cloned template and activate GitHub pages from the Branch "gh-pages". Every push to "main" will redeploy the Pages website. Test this in your local clone of the repo by running "make html" in the docs/ folder. 

Figures and Tables that are outputted from the scripts in the Scripts folder should be save in the docs/_static folder. References from the .rst files will refer to this folder.
